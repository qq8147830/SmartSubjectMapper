# 智能学科定位器部署配置指南

## 环境变量配置

在部署之前，您需要获取以下API密钥：

### 1. Supabase配置

**步骤1: 创建Supabase项目**
1. 访问 [https://supabase.com](https://supabase.com)
2. 注册/登录账号
3. 点击"New Project"创建新项目
4. 选择组织（个人或团队）
5. 填写项目信息：
   - 项目名称：`smart-subject-mapper`
   - 数据库密码：设置一个强密码（请保存）
   - 地区：选择 `Asia Pacific (Tokyo)` （距离较近）
6. 等待项目创建完成（约2-3分钟）

**步骤2: 获取API密钥**
1. 在Supabase项目仪表板中，点击左侧菜单的"Settings"
2. 点击"API"选项卡
3. 复制以下信息：
   - Project URL：`https://your-project-id.supabase.co`
   - anon/public key：以 `eyJ` 开头的长字符串

**步骤3: 配置数据库**
1. 在Supabase项目仪表板中，点击左侧菜单的"SQL Editor"
2. 点击"New Query"
3. 复制 `supabase/migrations/001_initial_schema.sql` 文件内容
4. 粘贴到查询编辑器中，点击"Run"

### 2. 通义千问API配置

**步骤1: 注册阿里云账号**
1. 访问 [https://www.aliyun.com](https://www.aliyun.com)
2. 注册/登录账号
3. 完成实名认证

**步骤2: 开通通义千问服务**
1. 访问 [https://dashscope.aliyun.com](https://dashscope.aliyun.com)
2. 在控制台中开通"通义千问"服务
3. 在"API-KEY管理"中创建新的API Key
4. 复制生成的API Key

## 环境变量设置

获取到所有密钥后，您有两种配置方式：

### 方式一：本地开发环境
在项目根目录的 `.env` 文件中更新：

```env
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
VITE_QWEN_API_KEY=your-qwen-api-key
```

### 方式二：Vercel部署环境变量
在Vercel部署时，系统会提示您设置环境变量：
1. `VITE_SUPABASE_URL`
2. `VITE_SUPABASE_ANON_KEY` 
3. `VITE_QWEN_API_KEY`

## 部署流程

### 自动部署（推荐）
运行以下命令：
```bash
npm run deploy
```

### 手动部署
1. 先登录Vercel：`npx vercel login`
2. 部署项目：`npx vercel --prod`

## 常见问题

### Q: 部署后页面显示空白
**A:** 检查浏览器控制台是否有JavaScript错误，通常是环境变量配置问题

### Q: 登录功能不工作
**A:** 确认Supabase项目已正确创建，且API密钥配置正确

### Q: AI分析功能报错
**A:** 验证通义千问API Key是否有效，且账户余额充足

### Q: 图表不显示
**A:** 检查ECharts库是否正常加载，刷新页面重试
