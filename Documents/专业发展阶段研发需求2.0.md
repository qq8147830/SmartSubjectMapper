# 专业发展阶段研发需求2.0

## 1. 需求背景

教师专业发展是教育行业的重要组成部分。通过构建科学的教师专业发展阶段模型，可以更好地评估教师的成长轨迹，为其提供个性化的发展建议和培训方案。本产品旨在建立一个智能化的教师专业发展阶段分析系统，帮助教育管理者和教师本人了解其专业发展水平和成长路径，结合《国培计划课程标准》《小学/中学教师专业标准（试行）》等政策标准，实现精准的教师专业发展阶段评估。

## 2. 产品目标

- 构建基于国家标准的教师专业发展阶段评估模型
- 提供个性化、可操作的专业发展建议
- 实现教师专业发展的可视化展示
- 支持多维度的专业能力分析与对比
- 生成标准化的教师数字画像分析报告

## 3. 功能需求

### 3.1 数据接入
- **教师基础信息接入**：接入教师姓名、教龄、地区、学科、职务、入职时间等基础信息
- **专业发展数据接入**：接入专业理念与师德、专业知识、专业技能、专业发展、数字素养等维度的评估数据
- **历史发展数据接入**：接入教师过往专业发展轨迹数据，包括历次评估时间、分数、趋势等
- **标准数据接入**：接入TeachSeek常模值、区域平均值、国家/地方标准值等对比标准数据
- **政策标准数据接入**：接入《国培计划课程标准》《小学/中学教师专业标准（试行）》等政策文件数据
- **培训记录数据接入**：接入教师培训历史、证书获取情况、学分累计等数据

### 3.2 数据清洗
- **数据格式标准化**：
  - 时间格式统一：将"2025.2"、"2025/2"、"2025-2"、"2025年2月"等格式统一转换为"YYYY年MM月"
  - 趋势标识标准化：将"上升"、"下降"、"持平"、"↑"、"↓"、"-"等统一为标准符号
  - 教龄格式转换：将"四年"、"四"、"肆"等中文数字转换为阿拉伯数字
  - 分数范围验证：确保分数在0-100分范围内，超出范围的数据进行提示修正
  - 学科分类标准化：将不同表述的学科名称统一为标准分类

- **缺失值处理**：
  - 识别必填字段缺失情况（如姓名、教龄、学科等）
  - 对缺失数据进行标识和分类处理
  - 提供缺失数据补充建议和提示

- **异常值检测**：
  - 识别逻辑冲突数据（如教龄大于工作年限）
  - 检测范围异常数据（如分数超出合理范围）
  - 发现关联缺失数据（如缺少必要的关联信息）

- **重复数据处理**：
  - 识别重复的教师信息记录
  - 去除重复的评估记录
  - 处理相似但不完全重复的数据

- **敏感信息脱敏**：
  - 对教师姓名等敏感信息进行模糊化处理
  - 保护教师隐私数据
  - 符合《个人信息保护法》和《数据安全法》要求

### 3.4 产品校验
- **准确性校验**：验证专业发展阶段评估的准确性，与人工评估结果进行对比验证
- **一致性校验**：确保各维度评估结果与整体发展阶段的逻辑一致性
- **对比有效性校验**：验证与常模值和区域平均值对比的合理性
- **建议相关性校验**：确保发展建议与教师当前阶段和能力状况的匹配度
- **政策合规性校验**：验证评估结果符合《国培计划课程标准》等政策要求
- **数据完整性校验**：验证输入数据的完整性和有效性
- **可视化准确性校验**：确保图表数据与分析结果的一致性

### 3.5 功能规则

#### 3.5.1 阶段判定规则
- **适应期-新手教师判定规则**：
  - IF 专业理念与师德、专业知识、专业技能等维度得分均低于区域平均值10分以上 THEN 阶段为适应期-新手教师
  - 适用场景：新入职教师或教学经验不足的教师

- **发展期-熟练教师判定规则**：
  - IF 各维度得分在区域平均值±10分范围内 THEN 阶段为发展期-熟练教师
  - 适用场景：具备基本教学能力，正在提升专业水平的教师

- **成熟期-骨干教师判定规则**：
  - IF 各维度得分超过区域平均值10分以上但低于常模值10分以内 THEN 阶段为成熟期-骨干教师
  - 适用场景：教学能力较强，具备一定专业影响力

- **稳定期-卓越教师判定规则**：
  - IF 各维度得分接近或超过常模值 THEN 阶段为稳定期-卓越教师
  - 适用场景：教学能力优秀，具有示范引领作用

- **创造期-教育家型判定规则**：
  - IF 在3个及以上维度表现卓越（超过常模值20分以上）且具有创新引领能力 THEN 阶段为创造期-教育家型
  - 适用场景：在教育领域有创新贡献，具有引领作用的专家型教师

#### 3.5.2 趋势分析规则
- **能力提升趋势识别规则**：
  - IF 某维度分数相比上次评估提升5分以上 THEN 标识为↑↑趋势（显著提升）
  - IF 某维度分数相比上次评估提升1-4分 THEN 标识为↑趋势（有所提升）
  - IF 某维度分数相比上次评估下降1-4分 THEN 标识为↓趋势（有所下降）
  - IF 某维度分数相比上次评估下降5分以上 THEN 标识为↓↓趋势（显著下降）
  - IF 某维度分数相比上次评估无明显变化（±0.5分） THEN 标识为-趋势

- **阶段性趋势分析规则**：
  - IF 连续3次评估显示同一维度持续提升 THEN 判定该维度进入上升通道
  - IF 连续3次评估显示同一维度持续下降 THEN 判定该维度需要重点关注

#### 3.5.3 可视化规则
- **路径图生成规则**：
  - 按5阶段模型生成线性路径图，从左到右依次为：适应期→发展期→成熟期→稳定期→创造期
  - 高亮显示当前阶段，使用不同颜色标识
  - 标注阶段转换的关键能力要求

- **柱状对比图生成规则**：
  - 生成5维度柱状对比图，包含个人数据、区域平均值、常模值
  - 使用不同颜色区分三种数据类型
  - 添加数值标签和对比标识

- **雷达图生成规则**：
  - 生成5维度雷达图，展示个人能力分布
  - 区域平均值和常模值作为参考线显示
  - 标注优势和劣势维度

#### 3.5.4 政策标准匹配规则
- **国培计划标准匹配规则**：
  - IF 教师能力评估结果符合《国培计划课程标准》中某类培训对象要求 THEN 推荐相应培训项目
  - 根据教师发展阶段匹配相应的国培计划模块

- **教师专业标准匹配规则**：
  - IF 教师在某维度能力低于《小学/中学教师专业标准（试行）》要求 THEN 标识为急需提升维度
  - 根据专业标准要求生成针对性发展建议
