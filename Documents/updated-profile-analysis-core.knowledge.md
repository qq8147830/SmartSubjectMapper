# 教师个人培训画像智能分析核心规范

## 画像体系构建规范（基于5.0-1提示词）

### 画像结构（1:1复刻原报告）

#### 1. 画像顶部：个人资料区域
- **展示内容**：教师姓名+基础信息（教龄: X年 | 地区: XX区 | 学科: XX | 职务: XX | 当前阶段: XX期-XX教师 | 已获培训学时: XX学时）
- **功能标识**：配套"导出报告""更新数据"操作按钮说明

#### 2. 我的专业发展路径
- **路径图展示**：从左到右呈现5个阶段（适应期-新手教师、发展期-熟练教师、成熟期-骨干教师、稳定期-卓越教师、创造期-教育家型），并标记当前所处阶段
- **多维度发展水平对比**：以柱状图呈现"专业理念与师德、专业知识、专业技能、专业发展、数字素养"5维度，对比TeachSeek常模值（紫色）、区域平均值（橙色）、我的数据值（绿色）

#### 3. 当前专业发展阶段分析
- **雷达图展示**：以"专业理念与师德、专业知识、专业技能、专业发展、数字素养"为维度，呈现我的数据值、区域平均值、常模值的多边形对比
- **细分指标列表**：每个维度下展示细分项（如专业理念与师德含"遵守师德与教育法规"等）、对应分数、趋势标识（↑/↓）

#### 4. 细分维度数据模块
- **专业发展**：展示"教研活动参与度、专业学习主动性、同伴互助与合作、教育研究能力"数据+趋势；配套"历年培训学时分布"折线图（含入职培训、第1年、第2年等节点）
- **数字素养**：展示"教育技术应用能力、在线教学平台使用、数字化教学资源开发、数据驱动教学改进"数据+趋势；配套"培训需求优先级"（高/中）标识

#### 5. 培训画像分析-2.0新版

##### （1）【培训历史智能评估模块】
- **培训项目列表**：展示培训名称、级别（国家级/市级/省级/区级/校本）、时间、学时、证书状态（已获得证书）、结业状态；支持按"专业理念与师德/专业知识"等类别、年份筛选
- **培训分布图表**：培训级别分布饼图（标注各级别项数占比）、培训维度分布图表（标注学生发展知识/学科知识等维度的培训项数占比）

##### （2）【智能决策分析模块】
- **评估结论**：基于培训记录的综合分析（如"学科知识维度较高，师德培训、教学设计待提升"）
- **推荐培训主题**：按优先级（高/中）展示主题、等级（高优先级/中优先级）、学时、提升方向
- **学科教研智能评估**：说明AI课例观察等工具的应用价值
- **周期学分要求**：展示"当年学分要求（不少于）/我已获得学分""省内X年累计学分要求（不少于）/我已获得学分"
- **教情分析对决策的策略和规则**：
  - **长板强化策略**：基于教情分析的长板信息，推荐强化该长板的高级培训课程，提供进阶培训、专业发展、导师角色等建议
  - **短板补强策略**：基于教情分析的短板信息，优先推荐补强该短板的基础培训课程，提供基础培训、技能提升、同伴互助等建议
  - **偏态校正策略**：基于教情分析的偏态量，提供平衡发展的培训规划，推荐平衡性培训、跨领域学习、综合能力提升等
  - **策略融合规则**：将教情分析的策略建议与现有培训推荐模型融合，生成融合教情分析建议的个性化培训方案
  - **知识维度决策规则**：
    - IF 教情分析显示知识维度存在短板（如"5以内数的认识和加减法"偏态量为+）THEN 推荐"学科知识教学策略"相关培训，优先级：高
    - IF 教情分析显示知识维度存在长板（如"认识图形"偏态量为-）THEN 策略建议："教研上需关注教师教学的方法策略"，推荐"教学方法优化"培训，优先级：中
  - **技能维度决策规则**：
    - IF 教情分析显示技能维度存在短板（如"推理、数学表达"偏态量为-）THEN 推荐"学生推理能力培养""数学表达能力提升"相关培训，优先级：高，策略："教师教学的成效好，只需有针对性的加强学生的学、练即可"
    - IF 教情分析显示技能维度存在长板（如"认知、运算、操作"偏态量为-）THEN 策略建议："教师的教学成效很好"，保持现有培训节奏，优先级：低
  - **能力素养维度决策规则**：
    - IF 教情分析显示能力素养维度存在短板（如"逻辑思维"偏态量为-）THEN 推荐"逻辑思维能力培养"相关培训，优先级：高，策略："教师要在如何提高整体逻辑思维的能力素养水平上下功夫"
    - IF 教情分析显示能力素养维度存在长板（如"直观想象、数理关系"偏态量为+/-）THEN 策略建议："教研上需关注数理关系的方法策略"，推荐"教学方法深化"培训，优先级：中
  - **综合决策规则**：
    - 基于教情分析的长板/短板数据，结合偏态量（+表示需要关注，-表示表现良好），生成针对性的培训推荐
    - 教情分析策略建议直接转化为培训推荐的方向和重点
    - 优先级判定：短板维度→高优先级，长板维度→中/低优先级
    - 培训主题匹配：将教情分析的策略建议（如"关注教学方法策略"）映射到具体的培训主题（如"教学方法创新与实践"）

##### （3）【教研能力智能评估模块】
- **评估项列表**：展示"AI课堂观察、微格教学评价、学业质量与能力素养评价"等项目的评估时间、等级（优秀/良好）、平均分、细分项分数（如课堂诊断XX分、实践操作XX分）
- **操作标识**：配套"查看评分分析报告"按钮说明

## 数据采集与处理规范

### 核心数据维度对照表

| **模块名称** | **核心数据项** | **采集来源** |
| --- | --- | --- |
| 教师基本信息 | 姓名、教龄、地区、学科、职务、当前专业发展阶段、历年已获培训学时 | 教师入职登记/年度信息表，来自《教师信息管理系统》 |
| 专业发展路径 | 专业发展阶段（5阶段）、各维度常模值（TeachSeek）、区域平均值、我的数据值 | 《教师专业发展路径及下级指标体系》+ 测评数据常模库 |
| 当前专业发展阶段分析 | 各维度细分项分数+趋势（↑/↓） | 专业发展量表诊断数据/课堂教学诊断表 |
| 专业发展维度 | 教研活动参与度、专业学习主动性、同伴互助与合作、教育研究能力的分数+趋势；历年培训学时分布 | 教情分析表/核心素养发展测评应用中年级报告的教情分析部分接口 |
| 教情分析数据 | 知识维度（认识图形、数的认识和加减法、位置等）、技能维度（认知、运算、操作、应用、推理、数学表达等）、能力素养维度（直观想象、数理关系、逻辑思维等）的长板/短板分析、偏态量、策略建议 | 教情分析表（knowledgebase/教情分析表.pdf），来自核心素养发展测评应用 |
| 数字素养维度 | 教育技术应用能力、在线教学平台使用、数字化教学资源开发、数据驱动教学改进的分数+趋势；培训需求优先级 | 数字素养测评数据 |
| 培训历史智能评估 | 培训项目（名称、级别、时间、学时、证书状态）；培训级别分布；培训维度分布 | 承培机构结业名单/证书数据库，来自教师信息管理系统培训管理模块/或数据导入 |
| 智能决策分析 | 推荐培训主题、优先级（高/中）、学时、提升方向；周期学分要求 | 智能推荐模型+区域学分管理系统 |
| 教研能力智能评估 | 评估项（AI课堂观察等）、评估时间、等级、平均分、细分项分数 | 课堂教学诊断分析表/循证教研系统，来自AI课堂诊断应用 |

### 数据处理流程

#### A. 数据接入与预处理
- **A1 数据校验**：自动检测各工作表的必填字段是否完整、数据格式是否合规
- **A2 数据清洗**：对教师敏感信息自动做模糊化处理；对重复数据自动去重
- **A3 数据关联**：自动调用知识库中的《教师专业发展路径及下级指标体系》《TeachSeek常模数据库》《区域教师能力平均值数据库》
- **A4 教情分析数据接入**：
  - 接入教情分析表数据（knowledgebase/教情分析表.pdf）
  - 提取知识维度数据：认识图形、数的认识和加减法、位置等维度的长板/短板、偏态量、策略建议
  - 提取技能维度数据：认知、运算、操作、应用、推理、数学表达等维度的长板/短板、偏态量、策略建议
  - 提取能力素养维度数据：直观想象、数理关系、逻辑思维等维度的长板/短板、偏态量、策略建议
  - 数据格式要求：长板/短板标识、偏态量（+/-）、策略建议文本

#### B. 分模块数据处理
- **B1 教师基本信息模块**：提取数据，自动填充报告顶部的个人资料栏
- **B2 专业发展路径模块**：从数据提取"当前专业发展阶段"，关联常模/区域数据生成对比图
- **B3 当前专业发展阶段分析模块**：生成多维度雷达图，自动提取各细分项的"分数+趋势"
- **B4 专业发展/数字素养维度模块**：提取分数与趋势，生成对应维度的文字指标栏和图表
- **B5 培训历史智能评估模块**：整理培训项目数据，生成项目列表和分布图表
- **B6 智能决策分析模块**：
  - 基于能力短板数据，调用推荐模型生成匹配的培训主题
  - 接入教情分析数据，提取知识维度、技能维度、能力素养维度的长板/短板、偏态量、策略建议
  - 应用教情分析决策规则：
    - 知识维度短板→推荐学科知识教学策略培训（高优先级）
    - 技能维度短板→推荐对应技能培养培训（高优先级）
    - 能力素养维度短板→推荐能力素养提升培训（高优先级）
    - 长板维度→推荐深化培训（中/低优先级）
  - 将教情分析的策略建议转化为具体的培训推荐方向和提升重点
- **B7 教研能力智能评估模块**：整理评估数据，生成评估项列表

#### C. 报告可视化与输出
- **C1 排版复刻**：严格按照原报告的模块顺序排版
- **C2 图表渲染**：自动匹配原报告的图表类型
- **C3 最终输出**：生成可直接查看的在线报告（支持"导出报告"操作，导出格式为PDF）

### 异常场景处理
- **数据不足**：若某模块数据不足，在报告对应区域显示"当前暂无XX数据，补充后可自动更新"的提示
- **数据异常**：若能力维度分数与常模/区域值差异过大，在报告中增加"该维度为当前薄弱项，建议优先参与对应培训"的标注

## 知识库依赖规范

### 必须查询知识库的场景
1. **政策合规性校验**：所有指标、分析严格遵循国家及地方教师培训政策标准
2. **专业维度标注**：依据《国培计划课程标准》《小学/中学教师专业标准（试行）》《国家教师教育课程标准》等政策文件进行标注
3. **数据采集要求**：参考《教育基础数据》《教育系统人员基础数据》等标准
4. **隐私保护合规**：遵守《个人信息保护法》《数据安全法》《教育数据安全管理办法》

### 知识库查询路径
- **知识库位置**：`888-PromptX/培训画像助手/knowledgebase`
- **政策文件**：查询对应政策文件的完整内容
- **评价标准**：参考评价指标体系文档
- **数据格式**：查看数据示例文件
