# 根据培训主题进行智能决策研发需求

## 1. 需求背景

在教师培训体系中，如何根据教师的具体情况和培训主题进行智能决策，推荐最适合的培训方案，是提升培训效果的关键。当前的培训推荐系统需要升级为智能化决策系统，能够基于教师的能力现状、培训历史、教情分析等多维度数据，进行精准的培训主题匹配和决策。

## 2. 产品目标

- 建立基于多维度数据的智能决策模型
- 实现个性化培训主题推荐
- 提供科学的培训优先级排序
- 优化培训资源配置和效果评估

## 3. 功能需求

### 3.1 数据接入
- **教师能力数据接入**：接入教师在专业理念与师德、专业知识、专业技能、专业发展、数字素养等维度的能力评估数据
- **培训历史数据接入**：接入教师过往参与的培训项目、级别、时间、学时、证书状态等信息
- **教情分析数据接入**：接入知识维度、技能维度、能力素养维度的长板/短板分析、偏态量、策略建议（来自knowledgebase/教情分析表.pdf）
- **培训资源数据接入**：接入培训主题、级别、时长、内容描述、适用对象等培训资源信息
- **政策标准数据接入**：接入《国培计划课程标准》《小学/中学教师专业标准（试行）》等政策标准数据
- **区域对比数据接入**：接入区域平均值、TeachSeek常模值等对比基准数据

### 3.2 数据清洗
- **数据格式标准化**：统一培训主题命名规范、级别标识、时间格式等
- **重复数据处理**：去除重复的培训记录和资源信息
- **缺失值处理**：对培训资源描述不完整的条目进行标识
- **一致性检查**：确保培训主题与教师能力维度的匹配性
- **时效性验证**：验证培训资源的时效性和适用性

### 3.3 AI提示词工程实现机制
- **能力差距分析提示词**：分析教师当前能力与目标能力之间的差距
- **培训需求识别提示词**：基于能力差距和教情分析，识别具体的培训需求
- **主题匹配提示词**：将教师需求与可用培训主题进行智能匹配
- **优先级排序提示词**：根据紧急程度和重要性对推荐的培训主题进行排序
- **决策解释提示词**：生成推荐决策的解释说明，提高透明度

### 3.4 产品校验
- **推荐准确性校验**：验证培训主题推荐与教师实际需求的匹配度
- **优先级合理性校验**：确保推荐优先级符合教师发展需要
- **决策一致性校验**：验证决策结果与输入数据的逻辑一致性
- **资源可用性校验**：确认推荐的培训资源确实可用
- **个性化程度校验**：验证推荐结果的个性化程度

### 3.5 功能规则
- **智能推荐规则**：
  - **长板强化策略**：
    - IF 教情分析显示某维度存在明显长板 THEN 推荐强化该长板的高级培训课程
    - 推荐方向：进阶培训、专业发展、导师角色等
    - 优先级：中等优先级
  - **短板补强策略**：
    - IF 教情分析显示某维度存在明显短板 THEN 优先推荐补强该短板的基础培训课程
    - 推荐方向：基础培训、技能提升、同伴互助等
    - 优先级：高优先级
  - **偏态校正策略**：
    - IF 教情分析偏态量显示某维度偏态明显（+/-）THEN 提供平衡发展的培训规划
    - 推荐方向：平衡性培训、跨领域学习、综合能力提升等
    - 优先级：根据偏态程度确定

- **决策逻辑规则**：
  - **知识维度决策规则**：
    - IF 教情分析显示知识维度存在短板（如"5以内数的认识和加减法"偏态量为+）THEN 推荐"学科知识教学策略"相关培训，优先级：高
    - IF 教情分析显示知识维度存在长板（如"认识图形"偏态量为-）THEN 策略建议："教研上需关注教师教学的方法策略"，推荐"教学方法优化"培训，优先级：中
  - **技能维度决策规则**：
    - IF 教情分析显示技能维度存在短板（如"推理、数学表达"偏态量为-）THEN 推荐"学生推理能力培养""数学表达能力提升"相关培训，优先级：高，策略："教师教学的成效好，只需有针对性的加强学生的学、练即可"
    - IF 教情分析显示技能维度存在长板（如"认知、运算、操作"偏态量为-）THEN 策略建议："教师的教学成效很好"，保持现有培训节奏，优先级：低
  - **能力素养维度决策规则**：
    - IF 教情分析显示能力素养维度存在短板（如"逻辑思维"偏态量为-）THEN 推荐"逻辑思维能力培养"相关培训，优先级：高，策略："教师要在如何提高整体逻辑思维的能力素养水平上下功夫"
    - IF 教情分析显示能力素养维度存在长板（如"直观想象、数理关系"偏态量为+/-）THEN 策略建议："教研上需关注数理关系的方法策略"，推荐"教学方法深化"培训，优先级：中

- **优先级判定规则**：
  - **高优先级**：能力短板、紧急需求、政策要求
  - **中优先级**：能力提升、专业发展、兴趣导向
  - **低优先级**：维持现状、巩固优势、非紧急需求

- **匹配度计算规则**：
  - 基于能力差距计算匹配度：差距越大，匹配度越高
  - 基于培训历史计算匹配度：未参与过的相关培训匹配度更高
  - 基于教情分析计算匹配度：与教情分析建议一致的培训匹配度更高

- **决策约束规则**：
  - IF 某培训项目已获得证书 THEN 不再推荐相同或相似培训
  - IF 培训学时要求与教师当前需求不匹配 THEN 调整推荐或提供说明
  - IF 培训级别与教师当前发展阶段不匹配 THEN 提供适合级别的替代推荐

- **效果预测规则**：
  - 基于历史数据预测培训效果
  - 考虑教师参与度和完成率
  - 预估培训后的能力提升程度

## 4. 技术要求

- 实现多维度数据融合分析算法
- 支持实时智能决策和批量决策
- 提供可解释的决策过程
- 确保决策模型的公平性和无偏性
- 支持决策规则的动态配置和优化

## 5. 预期成果

- 个性化的培训主题智能推荐
- 科学的培训优先级排序
- 可解释的决策过程
- 优化的培训资源配置
- 提升的培训效果和满意度
