# 根据培训主题进行智能决策研发需求2.0

## 1. 需求背景

在教师培训体系中，如何根据教师的具体情况和培训主题进行智能决策，推荐最适合的培训方案，是提升培训效果的关键。当前的培训推荐系统需要升级为智能化决策系统，能够基于教师的能力现状、培训历史、教情分析等多维度数据，并结合《国培计划课程标准》《小学/中学教师专业标准（试行）》等政策标准，进行精准的培训主题匹配和决策。系统需要实现从数据采集、培训数据清洗和标注，到智能决策分析的全流程自动化。

## 2. 产品目标

- 建立基于国家标准和多维度数据的智能决策模型
- 实现个性化、精准的培训主题推荐
- 提供科学的培训优先级排序和资源配置
- 优化培训效果评估和反馈机制
- 生成标准化的教师数字画像分析报告

## 3. 功能需求

### 3.1 数据接入
- **教师能力数据接入**：接入教师在专业理念与师德、专业知识、专业技能、专业发展、数字素养等维度的能力评估数据
- **培训历史数据接入**：接入教师过往参与的培训项目、级别、时间、学时、证书状态、结业状态等信息
- **教情分析数据接入**：接入知识维度、技能维度、能力素养维度的长板/短板分析、偏态量、策略建议（来自knowledgebase/教情分析表.pdf）
- **培训资源数据接入**：接入培训主题、级别、时长、内容描述、适用对象、培训方式、师资信息等培训资源信息
- **政策标准数据接入**：接入《国培计划课程标准》《小学/中学教师专业标准（试行）》《国家教师教育课程标准》等政策标准数据
- **区域对比数据接入**：接入区域平均值、TeachSeek常模值、优秀教师标准等对比基准数据
- **实时评估数据接入**：接入教师实时教学表现、学生反馈、同行评价等动态评估数据

### 3.2 数据清洗
- **数据格式标准化**：
  - 培训主题命名规范：统一培训主题的命名和分类标准
  - 级别标识标准化：统一国家级、省级、市级、区级、校级等标识
  - 时间格式统一：统一培训时间、报名时间、结业时间等格式
  - 学科分类标准化：统一学科分类体系
  - 评分标准统一：统一各项能力评分标准

- **重复数据处理**：
  - 去除重复的培训记录（基于教师ID+培训项目+时间的组合判断）
  - 去除重复的培训资源信息
  - 合并相似但不完全重复的培训项目

- **缺失值处理**：
  - 对培训资源描述不完整的条目进行标识
  - 基于相似培训项目补充缺失信息
  - 对关键字段缺失的记录进行特殊处理

- **一致性检查**：
  - 确保培训主题与教师能力维度的匹配性
  - 验证培训级别与教师发展阶段的适配性
  - 检查培训内容与政策标准的符合性

- **时效性验证**：
  - 验证培训资源的时效性和适用性
  - 检查培训项目的开课状态
  - 验证培训资源的更新频率

- **政策合规性验证**：
  - 验证培训内容是否符合《国培计划课程标准》
  - 检查培训项目是否符合教师专业发展要求
  - 确保培训资源符合教育政策规定

### 3.4 产品校验
- **推荐准确性校验**：验证培训主题推荐与教师实际需求的匹配度，准确率目标≥85%
- **优先级合理性校验**：确保推荐优先级符合教师发展需要和政策要求
- **决策一致性校验**：验证决策结果与输入数据的逻辑一致性
- **资源可用性校验**：确认推荐的培训资源确实可用且符合要求
- **个性化程度校验**：验证推荐结果的个性化程度和差异化水平
- **政策合规性校验**：确保推荐结果符合《国培计划课程标准》等政策要求
- **公平性校验**：验证决策结果对不同教师群体的公平性
- **可解释性校验**：确保决策过程和结果具有可解释性

### 3.5 功能规则

#### 3.5.1 智能推荐规则

##### 长板强化策略：
- **触发条件**：IF 教情分析显示某维度存在明显长板（分数高于区域平均值20分以上）
- **推荐逻辑**：推荐强化该长板的高级培训课程
- **推荐方向**：进阶培训、专业发展、导师角色、专家培养等
- **优先级**：中等优先级（在短板问题解决后）
- **政策匹配**：符合教师专业发展标准中的优秀教师培养要求

##### 短板补强策略：
- **触发条件**：IF 教情分析显示某维度存在明显短板（分数低于区域平均值15分以上）
- **推荐逻辑**：优先推荐补强该短板的基础培训课程
- **推荐方向**：基础培训、技能提升、同伴互助、专项辅导等
- **优先级**：高优先级（紧急需求）
- **政策匹配**：符合教师专业标准中的基本能力要求

##### 偏态校正策略：
- **触发条件**：IF 教情分析偏态量显示某维度偏态明显（+/-值超过0.5）
- **推荐逻辑**：提供平衡发展的培训规划
- **推荐方向**：平衡性培训、跨领域学习、综合能力提升等
- **优先级**：根据偏态程度确定（偏态量越大，优先级越高）
- **政策匹配**：符合教师全面发展的政策要求

##### 策略融合规则：
- **触发条件**：IF 教情分析提供具体策略建议
- **推荐逻辑**：将策略建议与现有培训推荐模型融合
- **推荐方向**：生成融合教情分析建议的个性化培训方案
- **优先级**：高优先级（针对性强的建议）
- **政策匹配**：结合国培计划标准进行优化

#### 3.5.2 决策逻辑规则

##### 知识维度决策规则：
- **知识短板识别规则**：
  - IF 教情分析显示知识维度存在短板（如"5以内数的认识和加减法"偏态量为+）AND 分数低于标准要求 THEN 推荐"学科知识教学策略"相关培训
  - 优先级：高
  - 政策匹配：符合《国培计划课程标准》中学科知识培训要求
  - 预期效果：提升学科知识掌握水平

- **知识长板应用规则**：
  - IF 教情分析显示知识维度存在长板（如"认识图形"偏态量为-）AND 分数达到优秀标准 THEN 策略建议："教研上需关注教师教学的方法策略"
  - 推荐"教学方法优化"培训
  - 优先级：中
  - 政策匹配：符合教师专业发展中的方法创新要求

##### 技能维度决策规则：
- **技能短板强化规则**：
  - IF 教情分析显示技能维度存在短板（如"推理、数学表达"偏态量为-）AND 影响教学效果 THEN 推荐"学生推理能力培养""数学表达能力提升"相关培训
  - 优先级：高
  - 策略："教师教学的成效好，只需有针对性的加强学生的学、练即可"
  - 政策匹配：符合《小学/中学教师专业标准》中教学技能要求

- **技能长板维持规则**：
  - IF 教情分析显示技能维度存在长板（如"认知、运算、操作"偏态量为-）AND 教学效果良好 THEN 策略建议："教师的教学成效很好"
  - 保持现有培训节奏
  - 优先级：低
  - 政策匹配：符合优秀教师保持优势的要求

##### 能力素养维度决策规则：
- **能力短板提升规则**：
  - IF 教情分析显示能力素养维度存在短板（如"逻辑思维"偏态量为-）AND 影响学生发展 THEN 推荐"逻辑思维能力培养"相关培训
  - 优先级：高
  - 策略："教师要在如何提高整体逻辑思维的能力素养水平上下功夫"
  - 政策匹配：符合《国家教师教育课程标准》中能力素养要求

- **能力长板深化规则**：
  - IF 教情分析显示能力素养维度存在长板（如"直观想象、数理关系"偏态量为+/-）AND 表现优秀 THEN 策略建议："教研上需关注数理关系的方法策略"
  - 推荐"教学方法深化"培训
  - 优先级：中
  - 政策匹配：符合专家型教师培养要求

#### 3.5.3 优先级判定规则
- **高优先级**：能力短板、紧急需求、政策强制要求、教学效果差、影响学生发展
- **中优先级**：能力提升、专业发展、兴趣导向、教学效果一般、个人发展目标
- **低优先级**：维持现状、巩固优势、非紧急需求、教学效果良好、可选性培训

#### 3.5.4 匹配度计算规则
- **基于能力差距的匹配度**：
  - 匹配度 = (能力差距分数 / 最大可能差距) * 0.4 + (政策符合度分数 / 最大政策分数) * 0.3 + (历史匹配度分数 / 最大历史分数) * 0.2 + (资源质量分数 / 最大质量分数) * 0.1
  - 差距越大，匹配度越高（但不超过政策符合度权重）

- **基于培训历史的匹配度**：
  - IF 培训项目已参与且通过 THEN 匹配度降低60%
  - IF 培训项目已参与但未通过 THEN 匹配度降低30%
  - IF 培训项目未参与 AND 内容相关 THEN 匹配度增加20%
  - IF 培训项目未参与 AND 高度相关 THEN 匹配度增加40%

- **基于教情分析的匹配度**：
  - IF 培训内容与教情分析建议完全一致 THEN 匹配度增加40%
  - IF 培训内容与教情分析建议高度一致 THEN 匹配度增加30%
  - IF 培训内容与教情分析建议部分一致 THEN 匹配度增加15%
  - IF 培训内容与教情分析建议不一致 THEN 匹配度降低10%

#### 3.5.5 决策约束规则
- **重复推荐约束**：
  - IF 某培训项目已获得证书 THEN 不再推荐相同或高度相似培训
  - IF 培训学时要求与教师当前需求不匹配 THEN 调整推荐或提供说明
  - IF 培训内容重复度超过80% THEN 仅推荐一次

- **级别匹配约束**：
  - IF 培训级别与教师当前发展阶段不匹配 THEN 提供适合级别的替代推荐
  - IF 培训内容难度与教师能力不匹配 THEN 调整推荐级别或提供前置培训建议

- **时间约束**：
  - IF 培训时间与教师已有培训时间冲突 THEN 提供时间调整建议
  - IF 培训频率过高（月内超过2次） THEN 建议分散安排
  - IF 培训时长与教师可用时间不匹配 THEN 推荐分阶段培训

- **政策合规约束**：
  - IF 推荐培训不符合《国培计划课程标准》 THEN 禁止推荐
  - IF 培训内容不符合教师专业标准 THEN 调整推荐
  - IF 培训资源不符合教育政策规定 THEN 不予推荐

#### 3.5.6 效果预测规则
- **基于历史数据的效果预测**：
  - IF 教师有相似培训历史 THEN 基于历史完成率和效果预测
  - 预测公式：预期效果 = 历史平均效果 × (1 + 个人能力系数) × (1 - 难度系数)

- **基于培训质量的效果预测**：
  - IF 培训资源质量评分 > 4.0 THEN 效果预测系数 × 1.2
  - IF 培训资源质量评分 > 3.0 THEN 效果预测系数 × 1.0
  - IF 培训资源质量评分 < 3.0 THEN 效果预测系数 × 0.8

- **基于参与度的效果预测**：
  - 考虑教师参与度历史数据
  - 预测完成率 = 历史平均完成率 × 培训兴趣系数 × 时间匹配系数

#### 3.5.7 个性化推荐规则
- **发展阶段适配**：
  - 适应期教师：重点推荐基础能力培训
  - 发展期教师：推荐能力提升和方法改进培训
  - 成熟期教师：推荐专业深化和创新培训
  - 稳定期教师：推荐引领示范和专家培养培训
  - 创造期教师：推荐研究创新和理论建构培训

- **学科特性适配**：
