# 培训画像2.0研发需求

## 1. 需求背景

教师培训画像系统是帮助教育管理者和教师本人了解培训需求、评估培训效果的重要工具。当前的培训画像系统需要升级到2.0版本，以提供更精准、更全面的培训分析和建议。新版本将整合更多数据源，特别是教情分析数据，以提供更深入的洞察。

## 2. 产品目标

- 实现更精准的教师培训需求分析
- 整合教情分析数据，提供教学实际情况洞察
- 优化智能决策分析，提供个性化培训建议
- 提升用户体验和报告可视化效果

## 3. 功能需求

### 3.1 数据接入
- **教师基础信息接入**：接入教师姓名、教龄、地区、学科、职务等基础信息
- **专业发展数据接入**：接入专业理念与师德、专业知识、专业技能等维度数据
- **教情分析数据接入**：接入知识维度、技能维度、能力素养维度的长板/短板分析、偏态量、策略建议（来自knowledgebase/教情分析表.pdf）
- **培训历史数据接入**：接入教师过往培训项目、级别、时间、学时、证书状态等信息
- **数字素养数据接入**：接入教育技术应用、在线教学、数字化资源开发等能力数据
- **教研能力数据接入**：接入AI课堂观察、微格教学评价等评估数据

### 3.2 数据清洗
- **数据格式标准化**：统一时间格式（YYYY年MM月）、趋势标识（↑/↓/-）、教龄格式（数字）、分数范围（0-100分）等
- **缺失值处理**：对缺失数据进行标识，显示"当前暂无XX数据，补充后可自动更新"
- **异常值检测**：识别超出合理范围的数据并进行提示修正
- **数据去重**：去除重复的培训记录和评估数据
- **敏感信息脱敏**：对教师敏感信息进行模糊化处理

### 3.3 AI提示词工程实现机制
- **综合分析提示词**：整合多维度数据，生成综合性的教师能力分析
- **教情分析解读提示词**：解读教情分析表中的长板/短板、偏态量、策略建议
- **智能推荐提示词**：基于教情分析和培训历史，生成个性化培训推荐
- **决策分析提示词**：应用长板强化、短板补强、偏态校正等策略生成培训建议
- **可视化生成提示词**：生成用于图表展示的描述和说明

### 3.4 产品校验
- **数据完整性校验**：验证各数据维度的完整性
- **逻辑一致性校验**：确保分析结果与输入数据的逻辑一致性
- **推荐合理性校验**：验证培训推荐与教师实际情况的匹配度
- **可视化准确性校验**：确保图表数据与分析结果的一致性
- **异常处理校验**：验证异常数据的处理和提示机制

### 3.5 功能规则
- **教情分析应用规则**：
  - **长板强化策略**：IF 教情分析显示某维度存在明显长板 THEN 推荐强化该长板的高级培训课程，提供进阶培训、专业发展、导师角色等建议
  - **短板补强策略**：IF 教情分析显示某维度存在明显短板 THEN 优先推荐补强该短板的基础培训课程，提供基础培训、技能提升、同伴互助等建议
  - **偏态校正策略**：IF 教情分析偏态量显示某维度偏态明显（+/-）THEN 提供平衡发展的培训规划，推荐平衡性培训、跨领域学习、综合能力提升等
  - **策略融合规则**：IF 教情分析提供具体策略建议 THEN 将策略建议与现有培训推荐模型融合，生成融合教情分析建议的个性化培训方案

- **智能决策规则**：
  - **知识维度决策规则**：
    - IF 教情分析显示知识维度存在短板（如"5以内数的认识和加减法"偏态量为+）THEN 推荐"学科知识教学策略"相关培训，优先级：高
    - IF 教情分析显示知识维度存在长板（如"认识图形"偏态量为-）THEN 策略建议："教研上需关注教师教学的方法策略"，推荐"教学方法优化"培训，优先级：中
  - **技能维度决策规则**：
    - IF 教情分析显示技能维度存在短板（如"推理、数学表达"偏态量为-）THEN 推荐"学生推理能力培养""数学表达能力提升"相关培训，优先级：高，策略："教师教学的成效好，只需有针对性的加强学生的学、练即可"
    - IF 教情分析显示技能维度存在长板（如"认知、运算、操作"偏态量为-）THEN 策略建议："教师的教学成效很好"，保持现有培训节奏，优先级：低
  - **能力素养维度决策规则**：
    - IF 教情分析显示能力素养维度存在短板（如"逻辑思维"偏态量为-）THEN 推荐"逻辑思维能力培养"相关培训，优先级：高，策略："教师要在如何提高整体逻辑思维的能力素养水平上下功夫"
    - IF 教情分析显示能力素养维度存在长板（如"直观想象、数理关系"偏态量为+/-）THEN 策略建议："教研上需关注数理关系的方法策略"，推荐"教学方法深化"培训，优先级：中

- **优先级判定规则**：
  - 短板维度 → 高优先级
  - 长板维度 → 中/低优先级
  - 能力急需提升维度 → 高优先级

- **可视化规则**：
  - 按固定模块顺序排版：个人信息→专业发展路径→当前阶段分析→各维度数据→培训画像→智能决策
  - 匹配原报告图表类型：路径图、柱状图、雷达图、饼图、折线图
  - 支持在线查看和PDF导出

## 4. 技术要求

- 支持PDF格式的教情分析表解析
- 实现高效的数据处理和分析算法
- 提供直观的可视化界面
- 确保数据安全和隐私保护
- 支持实时数据更新和动态报告生成

## 5. 预期成果

- 更精准的教师培训需求分析
- 整合教情分析数据的全面画像
- 个性化的培训推荐方案
- 优化的可视化报告展示
- 动态更新的培训跟踪机制
