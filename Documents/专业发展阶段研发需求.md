# 专业发展阶段研发需求

## 1. 需求背景

教师专业发展是教育行业的重要组成部分。通过构建科学的教师专业发展阶段模型，可以更好地评估教师的成长轨迹，为其提供个性化的发展建议和培训方案。本产品旨在建立一个智能化的教师专业发展阶段分析系统，帮助教育管理者和教师本人了解其专业发展水平和成长路径。

## 2. 产品目标

- 构建科学的教师专业发展阶段评估模型
- 提供个性化的专业发展建议
- 实现教师专业发展的可视化展示
- 支持多维度的专业能力分析

## 3. 功能需求

### 3.1 数据接入
- **教师基础信息接入**：接入教师姓名、教龄、地区、学科、职务等基础信息
- **专业发展数据接入**：接入专业理念与师德、专业知识、专业技能、专业发展、数字素养等维度的评估数据
- **历史发展数据接入**：接入教师过往专业发展轨迹数据
- **标准数据接入**：接入TeachSeek常模值、区域平均值等对比标准数据

### 3.2 数据清洗
- **数据格式标准化**：统一时间格式、分数范围、趋势标识等数据格式
- **缺失值处理**：对缺失数据进行标识和处理
- **异常值检测**：识别超出合理范围的数据并进行处理
- **数据去重**：去除重复的教师信息和评估记录

### 3.3 AI提示词工程实现机制
- **专业阶段评估提示词**：设计评估教师当前专业发展阶段的提示词，基于五阶段模型（适应期-新手教师、发展期-熟练教师、成熟期-骨干教师、稳定期-卓越教师、创造期-教育家型）
- **多维度对比分析提示词**：生成与常模值、区域平均值的对比分析
- **发展建议生成提示词**：根据教师当前阶段和能力短板，生成个性化发展建议
- **可视化描述提示词**：生成用于路径图、柱状图、雷达图等可视化元素的描述

### 3.4 产品校验
- **准确性校验**：验证专业发展阶段评估的准确性
- **一致性校验**：确保各维度评估结果与整体发展阶段的逻辑一致性
- **对比有效性校验**：验证与常模值和区域平均值对比的合理性
- **建议相关性校验**：确保发展建议与教师当前阶段和能力状况的匹配度

### 3.5 功能规则
- **阶段判定规则**：
  - IF 专业理念与师德、专业知识、专业技能等维度得分均低于区域平均值 THEN 阶段为适应期-新手教师
  - IF 各维度得分接近区域平均值 THEN 阶段为发展期-熟练教师
  - IF 各维度得分超过区域平均值但低于常模值 THEN 阶段为成熟期-骨干教师
  - IF 各维度得分接近或超过常模值 THEN 阶段为稳定期-卓越教师
  - IF 在多个维度表现卓越且具有创新引领能力 THEN 阶段为创造期-教育家型

- **趋势分析规则**：
  - IF 某维度分数相比上次评估上升 THEN 标识为↑趋势
  - IF 某维度分数相比上次评估下降 THEN 标识为↓趋势
  - IF 某维度分数相比上次评估无明显变化 THEN 标识为-趋势

- **可视化规则**：
  - 生成5阶段路径图，高亮显示当前阶段
  - 生成多维度柱状对比图，展示个人、区域、常模值对比
  - 生成多维度雷达图，展示个人能力分布

## 4. 技术要求

- 支持多种数据格式的接入
- 实现高效的数据处理和分析算法
- 提供直观的可视化界面
- 确保数据安全和隐私保护

## 5. 预期成果

- 教师专业发展阶段的准确评估
- 个性化发展建议的生成
- 可视化的发展路径展示
- 专业发展轨迹的动态跟踪
