## PromptX 背景介绍

PromptX 是一个基于 MCP（Model Context Protocol）协议的智能角色系统，能够通过自然语言交互激活和使用各种专业角色。每个角色都具备独特的专业领域知识和技能，可以为用户提供专业化的咨询服务。

### 核心特性

- **多角色系统**：支持多种专业角色，涵盖教育、战略、技术等领域
- **智能交互**：通过自然语言激活和使用角色
- **专业能力**：每个角色都有特定的领域专长和工作流程
- **模块化设计**：角色具备角色定义、思维模式、执行模式和知识库

## 系统配置

### MCP 服务器配置

```json
{
  "mcpServers": {
    "promptx": {
      "command": "npx",
      "args": ["-y", "@promptx/mcp-server"]
    }
  }
}
```

### 配置文件位置

- **全局配置**：`~/.promptx/config.json`
- **项目配置**：`.promptx/config.json`

## 常见问题

### Q1: 如何知道哪些角色已安装？

A: 可以查看 `~/.promptx/resource/role/` 目录，该目录包含所有已安装的角色。

### Q2: 角色激活后可以使用多长时间？

A: 角色激活后会持续工作，直到您激活其他角色或结束对话。

### Q3: 如何切换角色？

A: 直接使用新的激活命令即可切换到其他角色。

### Q4: 角色的回复质量如何保证？

A: 每个角色都具备完整的知识库、思维模式和执行流程，确保提供专业、准确的回复。

### Q5: 如何为角色提供反馈？

A: 可以直接在对话中向角色提供反馈，角色会根据上下文调整回复。
