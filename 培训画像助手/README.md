# 培训画像助手

## 项目简介

培训画像助手是一个专业的教师培训数据分析与画像生成系统，能够对教师培训数据进行智能处理、分析和可视化，生成个性化的培训画像报告。

## 功能特性

### 1. 数据处理
- ✅ 数据校验：检测必填字段完整性、格式合规性
- ✅ 数据清洗：脱敏、去重、标准化处理
- ✅ 异常检测：识别8类核心异常并生成异常提示清单
- ✅ 自动校正：对常见格式错误进行智能匹配和转换

### 2. 培训记录智能标注
- ✅ 维度标注：基于政策文件进行专业维度标注（一级/二级/三级维度）
- ✅ 层级类型识别：自动识别培训层级类型（国家级/省级/市级/区级/校级）
- ✅ 时间格式标准化：统一转换为"YYYY年MM月"格式
- ✅ 学段适配：根据学段差异进行维度适配

### 3. 数据分析
- ✅ 专业发展路径分析：五阶段发展路径可视化
- ✅ 多维度能力评估：专业理念与师德、专业知识、专业能力、专业发展、数字素养
- ✅ 培训历史统计：培训项目、学时、层级、维度分布分析
- ✅ 智能决策分析：推荐培训主题、优先级、学时建议

### 4. 报告生成
- ✅ 个性化培训画像报告：为每位教师生成HTML格式报告
- ✅ 汇总分析报告：整体数据统计和可视化分析
- ✅ 可视化图表：路径图、雷达图、柱状图、饼图等

## 使用说明

### 环境准备

1. 确保已安装Python 3.7+
2. 创建虚拟环境（如未创建）：
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # macOS/Linux
   # 或
   venv\Scripts\activate  # Windows
   ```

3. 安装依赖：
   ```bash
   pip install pandas
   ```

### 数据处理流程

#### 步骤1：数据校验和智能标注

```bash
python 数据处理.py
```

**功能：**
- 加载教师数据和培训记录数据
- 进行数据校验和异常检测
- 对培训记录进行智能标注
- 生成异常检测报告和数据校正记录

**输出文件：**
- `培训记录数据_已标注.csv` - 标注后的培训记录
- `异常检测报告.json` - 异常检测结果
- `数据校正记录表.csv` - 自动校正记录

#### 步骤2：生成培训画像报告

```bash
python 报告生成.py
```

**功能：**
- 为示例教师生成个性化培训画像报告

**输出文件：**
- `培训画像报告.html` - 单个教师培训画像报告

#### 步骤3：生成汇总分析报告

```bash
python 批量报告生成.py
```

**功能：**
- 生成整体数据汇总分析报告

**输出文件：**
- `培训画像汇总分析报告.html` - 整体汇总分析报告

### 报告内容说明

#### 个人培训画像报告包含：

1. **教师基本信息**
   - 姓名、教龄、地区、学校、学科、职务

2. **专业发展路径**
   - 五阶段发展路径图（适应期→发展期→成熟期→稳定期→创造期）
   - 多维度能力对比图（我的数据 vs 区域平均值 vs 常模值）

3. **当前专业发展阶段分析**
   - 多维度雷达图
   - 细分指标列表（含趋势标识）

4. **专业发展维度数据**
   - 教研活动参与度
   - 专业学习主动性
   - 累计培训学时
   - 历年培训学时分布图

5. **数字素养维度数据**
   - 教育技术应用能力
   - 在线教学平台使用
   - 数字化教学资源开发
   - 数据驱动教学改进

6. **培训历史智能评估**
   - 培训项目统计
   - 培训级别分布
   - 培训维度分布

7. **智能决策分析**
   - 推荐培训主题
   - 优先级建议
   - 建议学时
   - 提升方向

#### 汇总分析报告包含：

1. **总体统计**
   - 教师总数
   - 培训记录总数
   - 累计培训学时
   - 人均培训学时

2. **教龄分布**
3. **地区分布**
4. **学科分布**
5. **培训层级分布**
6. **培训维度分布**

## 数据格式要求

### 教师数据.csv

必填字段：
- `教师ID` - 教师唯一标识
- `姓名` - 教师姓名
- `教龄` - 数字格式（如：4）
- `所在地区` - 地区名称
- `所在学校` - 学校名称
- `任教学科` - 学科名称
- `职务` - 职务名称

### 培训记录数据.csv

必填字段：
- `培训记录ID` - 培训记录唯一标识
- `培训项目名称` - 培训项目名称
- `培训类型` - 标准选项：国培、省培、市县培、校本研修、其它
- `培训完成时间` - YYYY-MM-DD格式（如：2025-01-15）
- `学时` - 数字格式，最小单位0.5

## 异常处理

系统能够自动检测和处理以下8类异常：

1. **必填字段缺失** - 检测并提示缺失的必填字段
2. **数据格式错误** - 检测格式不符合要求的数据
3. **逻辑冲突** - 检测数据间的逻辑矛盾
4. **关联缺失** - 检测关联数据的缺失
5. **重复数据** - 自动去重并保留最新记录
6. **范围异常** - 检测超出合理范围的数据
7. **知识库关联失败** - 提示未匹配到标准选项的数据
8. **文件格式不支持** - 检测文件格式兼容性

## 智能标注规则

### 维度标注规则

基于培训项目名称的语义匹配：

- **专业理念与师德**：师德、师风、职业道德、教育法规
- **专业知识**：学科、核心考点、知识、课程标准
- **专业能力**：教学设计、教学创新、课堂教学、教学策略、作业设计、评价、班级管理、班主任、学生辅导、特殊学生、家校共育、科研、论文、研究、心理健康、心理辅导、校本课程、课程开发、跨学科、融合
- **数字素养**：信息技术、智慧课堂、STEM、数字化、在线教学、数据驱动

### 培训层级类型映射

- `国培` → `国家级培训（国培）`
- `省培` → `省级培训（省培）`
- `市县培` → `市级培训（市培）`
- `校本研修` → `校级培训（校培）`
- `其它` → `其它`
