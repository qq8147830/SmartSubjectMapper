培训画像分析-2.0新版

培训画像分析说明

亲爱的老师，"培训画像"是基于您已完成的培训记录生成的个人专属培训情况总结，根据培训数据分析，在右侧提供了"智能决策分析"，帮您清晰了解自己的培训轨迹、能力特点与成长方向～

一、培训画像是什么？

它是整合了您的培训历史记录、能力维度分布、学分完成进度、智能提升建议的综合视图，相当于您的"培训成长档案"，能直观看到自己在培训方面的优势、待提升点，以及后续的适配学习方向。

二、培训画像的核心板块说明

1. 培训历史概览（左侧区域）

这是您已完成的培训清单，可通过这些功能快速筛选查看：

  • 分类标签

    • 顶部的"专业理念与师德""专业知识"等标签，点击可只显示对应类别的培训（当前默认显示"全部"）；"专业知识"下的"学生发展知识"等子标签，是更细分的筛选维度。

  • 筛选/排序

    "年份"下拉框可选择查看某一年的培训；"排序"可调整培训列表的展示顺序（默认是"时间降序"，最新培训排在前）。

  • 单条培训信息

    • 每条记录会显示培训名称、级别（国家级/市级等）、培训时间、累计学时，以及"已获得证书"标识（代表您完成培训并取得证明）；点击右侧下拉箭头，可展开看培训详细内容。

    每条记录会显示培训名称、级别（国家级/市级等）、培训时间、累计学时，以及"已获得证书"标识（代表您完成培训并取得证明）；点击右侧下拉箭头，可展开看培训详细内容。

  • 规则

    仅展示您已完成并通过考核的培训，未完成的培训不会出现在这里。


2. 培训类型分布（左侧下方图表区）

这是您培训情况的"数据统计总结"：

  • 培训级别分布（饼图）

    展示不同级别培训的数量占比（比如您的市级培训有2项，占总培训的33.3%），图表下方会标注每项级别的具体数量。

  • 培训维度分布（柱状图）

    按教师专业能力维度（如"学生发展知识""学科知识"）统计培训数量（比如您有5项培训属于"学生发展知识"维度，占83.3%）；点击"展开全部维度"可查看更多能力维度的分布情况。

  • 规则

    数据会随您新完成的培训实时同步更新。


3. 智能决策分析（右侧上方区域）

这是基于您的培训记录给出的能力分析与提升建议：

  • 能力诊断

    系统结合《教师专业标准》等规范，指出您的优势（"学科知识"参与度较高）与待提升点（"师德培训""教学设计"参与较少）。

  • 推荐培训主题

    继教网AI根据您的短板，按"优先级"推荐适配培训：

      "高优先级"：对应核心短板（如"数字化教学资源开发与应用"），建议优先参与；

      "中优先级"：对应辅助提升点（如"跨学科整合教学能力提升"），可在高优先级培训完成后推进；

    每个推荐培训都会标注"对应提升能力""学时"，点击"去遴选"可直接报名。

  • 规则

    推荐主题会随您的新培训记录动态更新，点击"刷新"可获取最新推荐。


4. 学分进度追踪（右侧下方区域）

这是您的培训学分完成情况（学分是教师继续教育学时的核心考核指标）：

  • 当年学分要求

    显示"当年需修满的最低学分 / 您已获得的学分"（当前是20/4，即今年至少要修20学分，目前已得4学分）。

  • 省内5年累计学分要求

    显示"5年内需修满的最低学分 / 您已累计的学分"（当前是120/10，即5年总共至少修120学分，目前已累计10学分）。

  • 规则

    只有完成培训并通过考核后，学分才会同步到这里。


5. 教研能力关联评估（页面底部区域）

这是您的培训能力在教研实践中的体现，展示您参与过的教研评估记录（如"AI课堂观察"），包含评估时间、等级评定、分项得分，点击"查看评估分析报告"可获取具体的教研改进建议。

三、培训画像的功能规则

    1. 所有内容基于您已完成并记录在系统中的培训/评估；

    2. 智能推荐、数据统计会随您的新培训动态更新；

    3. 学分、证书等信息需完成培训考核后才会同步。
